Wielomiany na listach powiązanych dokumentacja

Tytuł projektu zaliczeniowego: Implementacja wielomianów na bazie list powiązanych. Działania na wielomianach: dodawanie, odejmowanie, mnożenie, obliczanie wartości wielomianu (algorytm Hornera)

 

1. Działanie programu:

Program z wykorzystaniem struktury danych jaką są listy powiązane implementuje podstawowe działania matematyczne dla wielomianów. Przy jego pomocy można dodać, odjąć i pomnożyć dwa wielomiany przez siebie oraz obliczyć ich wartość dla pewnego argumentu – wykonać algorytm Hornera oraz sprawdzić, czy wielomian jest równy zero lub czy dwa wielomiany są sobie równe. 

 

Przykładowo w wyniku działania programu dla konkretnych wielomianów:

• 𝑃(𝑥) = 7 + 1x + 2x2 + 3x3
• 𝑊(𝑥) = 3 + 4x + 2x2 + 3x3 + 4x4

 

Uzyskamy wyniki: 

𝑃(𝑥) + 𝑊(𝑥) = 10 + 5x + 4x2 + 6x3 + 4x4

𝑃(𝑥) - 𝑊(𝑥) = 4 - 3x

𝑃(𝑥) ∗ 𝑊(𝑥) = 21 + 31x + 24x2 + 40x3 + 47x4 + 16x5 + 17x6 + 12x7  

Wartość wielomianu 𝑃(𝑥) dla x=2: 41

3. Sposób uruchomienia programu:

Aby uruchomić program należy wykonać komendę̨: make run

Aby usunąć zbędne pliki wykonawcze z katalogu projekt możemy użyć komendy: make clean 

Aby usunąć zbędne pliki, a następnie zarchiwizować pliki znajdujące się w folderze nalezy uzyc komendy: make tar
